# Welcome to Programming Lightning: Intro to Payment Channels

Welcome! If you're interested in learning how Bitcoin's Lightning Network works, you've come to the right place. Programming Lightning is a comprehensive course that teaches you how the Lightning Network works by guiding you through building a payment channel from scratch.

By the end of this course, your implementation will even pass some of the main [BOLT 3 Test Vectors](https://github.com/lightning/bolts/blob/master/03-transactions.md#appendix-b-funding-transaction-test-vectors). Test Vectors are pre-defined inputs and expected outputs that help developers ensure their code aligns with the protocol specification. When your implementation passes these test vectors, it means your code is correct, and your implementation is on its way to being interoperable with other Lightning implementations such as LND, LDK, Eclair, and Core Lightning.

In this course, we'll build payment channels from the ground up, starting with our Lightning wallet. Once we have that foundation, we'll explore the simplest possible payment channel and examine its limitations. Then, step by step, we'll address each weakness and build toward a full, BOLT-compliant Lightning channel in all its glory.
